## Start the project
> /seavphov

```npm install```

```npm run dev```

>/backend

```composer install```

```php artisan serve```


## backend API


> **get** `/api/books`

> **get** `/api/books/{id}`

> **post** `/api/books`

> **put** `/api/books/{id}`

> **delete** `/api/books/{id}`
